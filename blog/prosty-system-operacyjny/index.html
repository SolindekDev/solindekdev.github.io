Jak stworzyć prosty system operacyjny
W dzisiejszym artykule przedstawie jak stworzyć prosty system operacyjny

System operacyjny potrzebuje bootloadera, kernela itp. my że piszemy dziś bardzo prosty system. Napiszemy tylko wejście systemu w Asseblerze dokładnie w Nasm (Netwide Assembler) Pierwsze warto pobrać Nasm, wejdź na ich stronę https://nasm.us/

wejdź w zakładkę Download



Następnie wybierz najnowszą wersje w moim przypadku będzie to 2.15.05



Wybierz odpowiedni system operacyjny w moim przypadku będzie to Windows więc wybiorę win64



I pobierz installator. Co do linuxa jest tu trudniej bo jest wiele wersji do wybrania ale osoby z linuxem będą wiedzieć którą wybrać



Po kliknięciu powinnien się installować installator



Następnie będzie typowy dla każdego z systemów czas installacji programu. Po zainstalowaniu otwieramy nasz text edytor w moim przypdaku używam Visual Studio Code ale to zależy od was, tworzymy nowy folder ja nazwę go "PoradnikOS" i tworzymy pierwszy plik o nazwię "main.asm" . W pliku main.asm wpisujemy poniżej podany kod

Assembly Code
BITS 64 ; Ustawiamy kod jako 64 bitowy
ORG 0x7c00 ; Ustaw punkt początkowy na 0x7c00

jmp main ; Skocz do funkcji main

main: ; Main funckja
jmp $ ; Stwórz nieskończoną pętle

times 510 - ($-$$) db 0 ; Pamięć
dw 0xAA55 ; Sygnatura bootloadera

Dzięki kodu u góry możemy zrobić nieskończoną pętle systemu, dosłownie system będzie tworzył pętle do póki go nie wyłączysz

Teraz musimy kompilować nasz kod assemblera do kodu binarnego, Aby to zrobić musismy wpisać w CMD (Terminal):

nasm -f bin main.asm -o os.bin


Teraz fajnie by było odpalić taki system tylko jak to zrobić? Mamy wykorzystać do tego swój komputer? No nie jest taki specjalny program jak QEMU który trzeba pobrać ze strony https://www.qemu.org/download/ wybierzcie odpowiedni system i przejdzcie przez typowy system installacji

Teraz odpalmy! Odpal CMD (Terminal) i wpiszcie tak:

Windows:

"C:\Program Files\qemu\qemu-system-x86_64.exe" os.bin

Linux:

qemu-system-x86_64 os.bin
